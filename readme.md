SheepMusic
==========

Simple PDF music sheets viewer for normal PC and touch screens.

2024-2025 Gideon van der Kolf, noedigcode@gmail.com

Requirements:

- Qt5
- libQt5Pdf (On Ubuntu, libqt5pdf5 for running and qtpdf5-dev for building)

Building:
---------

Open the .pro file with QtCreator and build

Or run the following from the command line:
```
mkdir build
cd build
qmake ../sheepmusic.pro
make
```
