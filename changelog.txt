SheepMusic Changelog
====================

[1.0.2] - April 2025
--------------------

Changed

- When adding documents, add after current/selected document.

Fixed

- Fixed bug where cropping multiple pages didn't update page view when exiting
  crop mode.


[1.0.1] - 13 April 2025
-----------------------

Added

- Document list now have numbers.

Fixed

- Crash when adding documents (erroneously tried to set pen of crop rectangle
  instead of page rectangle).


[1.0.0] - 17 February 2025
--------------------------

Changed

- Change name to SheepMusic
- Keep zoom rectangle visible when zoomed. Rectangle no longer filled.
- Show page cropped border permanently (hepls when zooming in to draw).

Added

- Add about screen
- Add sheep logo

Fixed

- Alow drawing zoom rectangle in any direction.


[0.0.10] - 12 October 2024
--------------------------

Changed

- Window title: show modified star at the start and show session name last.
- When a file path in a session doesn't exist, attempt to load the document
  relative to the session path.

Added

- Console button on settings screen
- Safe file loading and saving.


[0.0.9] - 7 October 2024
------------------------

Added

- Zooming in draw mode


[0.0.8] - 7 September 2024
